# Sayın Dr. Şadi İdem — Klinik Web Sitesi

**Sayın Dr. Şadi İdem** için hazırlanan resmi web sitesi; çocuk sağlığı ve hastalıkları alanındaki uzmanlık, klinik hizmetler ve hasta iletişimini tek bir çevrimiçi vitrinde toplar.

Eskişehir’de uzun yıllara dayanan deneyimle; yenidoğan takibi, çocuk gelişimi, alerji ve astım gibi konularda bilgilendirici içerik sunar. Ziyaretçiler hizmetler ve blog yazıları üzerinden bilgi alabilir; randevu talebi formu ile kliniğe kolayca ulaşabilir. Türkçe ve İngilizce dil desteği, güncel içerik yönetimi ve yönetim paneli ile site günlük kullanıma uygundur.
